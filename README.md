# aws-deployer
utility image to process deployments to Amazon AWS EKS via CI/CD pipeline

Based on Ubuntu 18.04 with the following additional packages:

 - curl
 - ca-certificates
 - gnupg
 - python=2.7.15
 - python-pip=9.0.1*
 - python-dev=2.7.15*
 - build-essential
 - groff
 - nano
 - git
 - jq
 - apt-transport-https
 - software-properties-common
 - docker-ce=18.06.1*
 - awscli
 - aws-iam-authenticator
 - helm
